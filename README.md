# Chat App

A real-time chat application that allows users to communicate instantly. This app supports features such as direct messaging, group chats, and real-time notifications.

## Features

- User Authentication (Login/Signup)
- Real-time Direct Messaging
s
- Notifications (New messages, online status)




## Tech Stack

- **Frontend**: React.js, Redux, Socket.io, Tailwind CSS (or Material UI)
- **Backend**: Node.js, Express.js, Socket.io
- **Database**: MongoDB (with Mongoose)
